# Weather
## О проекте
Сервис по получении погоды по городам, с сохранение и фильтрацией запросов
## Требования
- docker, docker-compose
## Уставонка
- скачать репозиторий
- запустить терминал в папке репозитория
- скомпилитовать docker-compose образ `docker-compose build`
- запустить образ `docker-compose up`
- интерфейс в сервису доступен по адресу `http://127.0.0.1:8000/doc`
